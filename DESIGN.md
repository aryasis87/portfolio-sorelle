# Sorelle — Design System (Soft Bento Creative Portfolio)

> Concept: **soft pastel "bento" creative** — portofolio brand/visual designer yang playful-premium: lembut, membulat, layout bento modular, ceria tapi rapi. Platform: responsive web. Bahasa: ID/EN.

## Brand voice
Hangat, ramah, kreatif. "Crafting brands with heart." Personal & approachable.

## Color tokens
| Token | Hex | Pakai |
|---|---|---|
| `bg` | `#faf7f5` | latar off-white |
| `surface` | `#ffffff` | kartu |
| `ink` | `#2c2638` | teks utama |
| `muted` | `#857d92` | teks sekunder |
| `primary` | `#7c6cf0` | lavender (aksen utama) |
| `peach` | `#ffb59e` | aksen hangat |
| `mint` | `#9be3c8` | aksen segar |
| `butter` | `#ffe08a` | aksen ceria |
| `border` | `#ece6f0` | garis |
Kartu bento boleh pakai tint pastel berbeda (lavender/peach/mint/butter) untuk variasi.

## Typography
- Display: **Clash Display** atau **Quicksand**/**Poppins** (600–700) — judul membulat & ramah.
- Body: **Plus Jakarta Sans** / **Nunito** (400–600).

## Shape & elevation
Radius BESAR & ramah (20–28px), kartu bento penuh karakter; bayangan lembut berwarna (pastel); banyak rounded.

## Sections & functional complexity
1. **Hero bento**: kartu-kartu modular (foto, nama+role, status "open for work", sosial, mini-stat) tersusun grid bento.
2. **About** singkat + **animated stats**.
3. **Services** kartu pastel beraneka tint (hover lift).
4. **Recent Work / Portfolio**: **bento gallery** ukuran beragam + hover reveal judul/kategori + **detail modal** + (opsional) **filter kategori**.
5. **Brands** logo marquee.
6. **Testimonials** slider lucu.
7. **Contact**/CTA "Let's work together" + form.
Tambahan UX: reveal-on-scroll bertahap, hover tilt halus pada kartu, sticky nav, smooth-scroll, reduced-motion, a11y.

## Motion
Playful & lembut: bouncy hover (spring), tilt 3D ringan, stagger reveal, marquee, modal pop. 200–400ms.

## Layout
Bento grid responsif (kolom beragam → 1 kolom di mobile). Container ~1120px. Nav pill membulat.
